## 生图小贴士

1. **文字问题**：AI生成中文/英文会不准确，出来的是"假文字"装饰效果。这是正常的，出图后用PS/Figma叠加上真实文字就行。
2. **推荐画幅比例**：`16:9` 或 `3:2`，适合三屏横排展示。
3. **推荐工具**：
   - Midjourney v6：细节最丰富
   - 即梦/可灵：中文提示词理解好，出图快
   - Stable Diffusion + ControlNet：有参考图的话可以用Canny控制构图
