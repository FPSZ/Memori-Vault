## 重要提醒

**目前没有任何生图AI能生成清晰可读的中文/英文。** 文字崩坏是正常现象，不是你的提示词问题。

**正确 workflow：**
1. 用上面的提示词生出干净的UI框架图（文字区域是空白或装饰性乱码）
2. 用 Figma / Photoshop / 即时设计 在对应位置加上真实文字
3. 推荐字体：中文用思源宋体/阿里巴巴普惠体，英文用 Inter / Playfair Display

如果一定要AI生成"看起来像字的装饰"，可以在提示词里加：`decorative text-like symbols, stylized glyphs as design elements, unreadable character shapes` —— 这样AI会生成好看的假字当装饰，而不是强行写崩坏的真字。
