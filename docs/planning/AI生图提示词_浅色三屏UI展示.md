## 生图小贴士

1. **文字问题**：AI生成中文/英文会不准确，出来的是装饰性的"假文字"。这是正常的，出图后用PS/Figma叠加上真实文字即可。
2. **推荐画幅比例**：`16:9` 或 `3:2`，适合三屏横排展示。
3. **浅色图后期建议**：浅色背景容易显得平淡，出图后可以适当增加卡片阴影层次、调整标题字体粗细来提升质感。
