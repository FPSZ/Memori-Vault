总体架构

基于 Rust + SQLite + 混合检索 + 分层记忆 + MCP 的本地可信智能检索系统

系统内核

  • Rust 异步运行时（Tokio）
  • Axum HTTP 服务器 + WebSocket
  • Tauri 桌面端跨平台封装
  • React 前端 + TypeScript 类型安全
  • 模块化插件架构（mcp / tools / graph）

存储与索引

  • SQLite 主存储（文档、分块、图谱）
  • 本地向量索引（HNSW 近似检索）
  • Graph 边表（chunk_nodes 关系网络）
  • 全文检索（FTS5 关键词索引）
  • 智能分块（语义边界识别）

检索与推理

  • 语义检索（向量相似度 Top-K）
  • 关键词检索（BM25 + FTS5）
  • 混合检索（语义 × 关键词融合排序）
  • Graph-RAG（子图激活与推理）
  • 分层记忆检索（STM / MTM / LTM）

治理与扩展

  • MCP 协议服务（Tools / Resources / Prompts）
  • Agent 工具接口（记忆读写 + 语义搜索）
  • 企业策略引擎（审计日志 + 访问控制）
  • 模型生命周期管理（本地/远程切换）
  • 设置持久化与配置版本控制
