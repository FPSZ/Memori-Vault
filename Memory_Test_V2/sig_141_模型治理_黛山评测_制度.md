# 星衡智能模型治理内部制度：黛山评测（MOD-23）

发文：模型治理组　责任人：卢屿

第一条　适用范围。本制度仅适用于 黛山评测 项目（代号 MOD-23）相关团队，外部通用做法不作为本项目执行依据。另有同事反馈测试环境偶发抖动，已转运维单独跟进。

第二条　核心边界。黛山评测把模型可上线定义为离线对齐分不低于 0.82 且越权拒答率不低于 0.97。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。每次换底模必须先跑红队套件，再跑业务回归，最后才灰度。各环节负责人需留痕。插一句无关的：饮水机换了供应商，口感大家自行适应。

第四条　历史参数。对齐分上线门槛早期暂按 0.78 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由卢屿（模型治理组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要用公开榜单分数代替内部对齐分。
