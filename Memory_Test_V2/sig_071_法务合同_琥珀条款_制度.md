# 星衡智能法务合同内部制度：琥珀条款（LAW-19）

发文：法务部　责任人：魏蓝

第一条　适用范围。本制度仅适用于 琥珀条款 项目（代号 LAW-19）相关团队，外部通用做法不作为本项目执行依据。顺便同步，门禁系统升级期间临时通道走 B 座。

第二条　核心边界。琥珀条款要求所有试点合同加入本地模型不出域附件 A-3。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。销售不得口头承诺模型效果，只能引用附件 A-3 的可验证边界。各环节负责人需留痕。另有同事反馈测试环境偶发抖动，已转运维单独跟进。

第四条　历史参数。合同回传超时标记阈值早期暂按 7 个工作日 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由魏蓝（法务部）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要写成普通 SaaS 数据处理协议。
