# 星衡智能合规审查内部制度：青简合规（COMP-66）

发文：合规审查组　责任人：祁年

第一条　适用范围。本制度仅适用于 青简合规 项目（代号 COMP-66）相关团队，外部通用做法不作为本项目执行依据。顺便同步，门禁系统升级期间临时通道走 B 座。

第二条　核心边界。青简合规规定跨境调用必须留存调用方真实主体，匿名调用一律拒绝。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。新接口上线必须先过数据分级，再过出境评估，最后才开放。各环节负责人需留痕。另有同事反馈测试环境偶发抖动，已转运维单独跟进。

第四条　历史参数。日志最短留存天数早期暂按 90 天 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由祁年（合规审查组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用 90 天日志留存回答。
