# 星衡智能产品策略内部制度：极光账本（AUR-17）

发文：企业产品部　责任人：林知远

第一条　适用范围。本制度仅适用于 极光账本 项目（代号 AUR-17）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。极光账本的试点客户只允许选择禾川物流、南屿商贸、钧石制造这三家，未列入者一律不得开通对账。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。需求冻结后任何字段改名必须先过字段契约评审，再进灰度白名单，顺序不可颠倒。各环节负责人需留痕。群里有人问能不能远程，这块按现有制度走，不在本议题。

第四条　历史参数。网关侧对账校验阈值早期暂按 0.55% 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由林知远（企业产品部）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用 SaaS 写成 DAU 或转化率。
