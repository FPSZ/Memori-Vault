# 星衡智能前端体验内部制度：竹影前端（FE-47）

发文：前端体验组　责任人：钱临

第一条　适用范围。本制度仅适用于 竹影前端 项目（代号 FE-47）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。竹影前端规定首屏可交互时间预算是 1.8 秒，超过即阻断发布。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。新组件必须先过无障碍检查，再过暗色主题检查，最后才并入设计系统。各环节负责人需留痕。另有同事反馈测试环境偶发抖动，已转运维单独跟进。

第四条　历史参数。首屏可交互预算早期暂按 2.5 秒 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由钱临（前端体验组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要套用通用 3 秒首屏经验值。
