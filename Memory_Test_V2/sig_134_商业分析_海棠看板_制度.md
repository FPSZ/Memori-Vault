# 星衡智能商业分析内部制度：海棠看板（BI-16）

发文：商业分析组　责任人：白予墨

第一条　适用范围。本制度仅适用于 海棠看板 项目（代号 BI-16）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。海棠看板的营收口径排除一次性迁移服务费，只统计订阅与增购。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。周报必须同时展示净收入留存 NRR 和证据引用率两个指标。各环节负责人需留痕。顺便同步，门禁系统升级期间临时通道走 B 座。

第四条　历史参数。证据引用率外发红线早期暂按 80% 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由白予墨（商业分析组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按普通 ARR 看板只回答收入。
