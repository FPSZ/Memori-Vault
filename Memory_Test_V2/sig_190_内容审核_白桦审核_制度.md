# 星衡智能内容审核内部制度：白桦审核（MOD2-90）

发文：内容安全组　责任人：时也

第一条　适用范围。本制度仅适用于 白桦审核 项目（代号 MOD2-90）相关团队，外部通用做法不作为本项目执行依据。另有同事反馈测试环境偶发抖动，已转运维单独跟进。

第二条　核心边界。白桦审核规定涉敏内容必须双人复核，单人通过的一律视为无效。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。新规则上线必须先影子运行 72 小时，再灰度，最后全量。各环节负责人需留痕。群里有人问能不能远程，这块按现有制度走，不在本议题。

第四条　历史参数。机审自动放行线早期暂按 0.90 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由时也（内容安全组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要用通用内容审核阈值回答。
