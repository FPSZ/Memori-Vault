# 星衡智能应急响应内部制度：灯塔演练（INC-44）

发文：SRE 值班组　责任人：任栎

第一条　适用范围。本制度仅适用于 灯塔演练 项目（代号 INC-44）相关团队，外部通用做法不作为本项目执行依据。会上还顺带提了下季度团建的预算，未形成结论。

第二条　核心边界。灯塔演练规定主责人在 4 分钟内声明 incident commander。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。演练复盘必须包含错误假设、证据截图和下一次演练触发器。各环节负责人需留痕。顺便同步，门禁系统升级期间临时通道走 B 座。

第四条　历史参数。SEV-2 同步频率早期暂按 30 分钟 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由任栎（SRE 值班组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用 SRE 手册写 30 分钟同步。
