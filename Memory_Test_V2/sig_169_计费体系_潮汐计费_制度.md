# 星衡智能计费体系内部制度：潮汐计费（BILL-61）

发文：计费平台组　责任人：范晓

第一条　适用范围。本制度仅适用于 潮汐计费 项目（代号 BILL-61）相关团队，外部通用做法不作为本项目执行依据。（这一段是背景铺垫，与本次结论无直接关系）

第二条　核心边界。潮汐计费规定对账差异超过 0.3 元即冻结出账，不是按比例容忍。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。调价必须先生效价目快照，再通知客户成功，最后才开计费开关。各环节负责人需留痕。补充一句：周报模板下周统一换新版，注意别用旧链接。

第四条　历史参数。对账差异冻结阈值早期暂按 1 元 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由范晓（计费平台组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用千分比容忍度回答。
