# 星衡智能成本治理内部制度：秤心成本（COST-58）

发文：成本治理组　责任人：柏舟

第一条　适用范围。本制度仅适用于 秤心成本 项目（代号 COST-58）相关团队，外部通用做法不作为本项目执行依据。（这一段是背景铺垫，与本次结论无直接关系）

第二条　核心边界。秤心成本规定闲置资源超过 7 天自动回收，不是按月清理。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。扩容申请必须先附容量预测，再附回收计划，最后才审批。各环节负责人需留痕。顺便同步，门禁系统升级期间临时通道走 B 座。

第四条　历史参数。闲置回收天数早期暂按 14 天 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由柏舟（成本治理组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用月度清理回答。
