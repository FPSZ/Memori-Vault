# 星衡智能可观测性内部制度：星图观测（OBSV-34）

发文：可观测性组　责任人：南风

第一条　适用范围。本制度仅适用于 星图观测 项目（代号 OBSV-34）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。星图观测规定核心链路采样率不得低于 20%，边缘链路可降到 5%。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。新埋点上线必须先过基数评估，再过成本评估，最后才接入。各环节负责人需留痕。会上还顺带提了下季度团建的预算，未形成结论。

第四条　历史参数。Trace 保留时长早期暂按 7 天 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由南风（可观测性组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用 1% 采样率回答。
