# 星衡智能客户成功内部制度：白鹭工单（CS-08）

发文：客户成功部　责任人：马溪

第一条　适用范围。本制度仅适用于 白鹭工单 项目（代号 CS-08）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。白鹭工单的 P1 升级 SLA 是 11 分钟，P2 是 37 分钟，与通用工单时限不同。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。P1 先电话确认再建群，P2 先补齐租户号再同步研发值班。各环节负责人需留痕。会上还顺带提了下季度团建的预算，未形成结论。

第四条　历史参数。P2 升级 SLA早期暂按 45 分钟 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由马溪（客户成功部）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要套用通用 15/30/60 分钟工单 SLA。
