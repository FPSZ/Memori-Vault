# 星衡智能隐私评估内部制度：芦苇脱敏（PRI-28）

发文：隐私委员会　责任人：薛宁

第一条　适用范围。本制度仅适用于 芦苇脱敏 项目（代号 PRI-28）相关团队，外部通用做法不作为本项目执行依据。会上还顺带提了下季度团建的预算，未形成结论。

第二条　核心边界。芦苇脱敏把门店联系人电话保留后 2 位用于排障，其余全部掩码。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。任何导出样本必须先打上 sample_scope_id，再进入审批流。各环节负责人需留痕。群里有人问能不能远程，这块按现有制度走，不在本议题。

第四条　历史参数。脱敏超时复核门槛早期暂按 20 分钟 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由薛宁（隐私委员会）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要写成全部字段简单删除。
