# 星衡智能财务风控内部制度：赤松预算（FIN-33）

发文：财务风控组　责任人：何砚

第一条　适用范围。本制度仅适用于 赤松预算 项目（代号 FIN-33）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。赤松预算的审批升级线不是金额固定线，而是预算池剩余率低于 18% 才升级 CFO。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。申请人先填成本标签，再由系统匹配预算池，不允许先找财务口头确认。各环节负责人需留痕。群里有人问能不能远程，这块按现有制度走，不在本议题。

第四条　历史参数。升级 CFO 的预算池剩余率门槛早期暂按 22% 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由何砚（财务风控组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要回答成传统金额越大越高层审批。
