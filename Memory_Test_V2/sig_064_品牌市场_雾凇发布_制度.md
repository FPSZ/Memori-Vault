# 星衡智能品牌市场内部制度：雾凇发布（MKT-05）

发文：品牌市场部　责任人：沈乔

第一条　适用范围。本制度仅适用于 雾凇发布 项目（代号 MKT-05）相关团队，外部通用做法不作为本项目执行依据。顺便同步，门禁系统升级期间临时通道走 B 座。

第二条　核心边界。雾凇发布的禁用词清单包含颠覆行业、零成本替代、全自动合规三个词。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。对外物料先过法务术语检查，再过品牌语气检查，顺序不能反。各环节负责人需留痕。另有同事反馈测试环境偶发抖动，已转运维单独跟进。

第四条　历史参数。闭门演示预约目标早期暂按 15 家 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由沈乔（品牌市场部）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要输出营销通用夸张词。
