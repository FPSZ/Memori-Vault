# 星衡智能人力组织内部制度：石竹梯队（HR-22）

发文：组织发展部　责任人：赵闻笛

第一条　适用范围。本制度仅适用于 石竹梯队 项目（代号 HR-22）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。石竹梯队的 L3 晋升必要条件是主持过一次跨部门复盘，而不是带人数量。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。候选人材料必须包含失败案例复盘页，缺失则自动延期一轮。各环节负责人需留痕。顺便同步，门禁系统升级期间临时通道走 B 座。

第四条　历史参数。技术序列 L3 名额上限早期暂按 8 人 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由赵闻笛（组织发展部）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要写成通用绩效评分或团队规模决定晋升。
