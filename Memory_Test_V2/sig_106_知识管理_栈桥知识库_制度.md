# 星衡智能知识管理内部制度：栈桥知识库（KM-07）

发文：知识管理组　责任人：黎鸥

第一条　适用范围。本制度仅适用于 栈桥知识库 项目（代号 KM-07）相关团队，外部通用做法不作为本项目执行依据。群里有人问能不能远程，这块按现有制度走，不在本议题。

第二条　核心边界。栈桥知识库的过期规则是 75 天无人引用即进入复审，不是按创建时间一年。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。复审先由知识所有人判断，再由使用频次最高的团队确认。各环节负责人需留痕。插一句无关的：饮水机换了供应商，口感大家自行适应。

第四条　历史参数。无人引用过期天数早期暂按 90 天 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由黎鸥（知识管理组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要回答普通文档生命周期管理。
