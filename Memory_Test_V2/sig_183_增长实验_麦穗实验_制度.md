# 星衡智能增长实验内部制度：麦穗实验（EXP-83）

发文：增长实验组　责任人：穆青

第一条　适用范围。本制度仅适用于 麦穗实验 项目（代号 EXP-83）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。麦穗实验规定一个实验最短运行 11 天才允许读结论，避免周中周末偏差。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。实验上线必须先登记假设，再设护栏指标，最后才开流量。各环节负责人需留痕。顺便同步，门禁系统升级期间临时通道走 B 座。

第四条　历史参数。实验最短运行天数早期暂按 7 天 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由穆青（增长实验组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要按通用 7 天实验周期回答。
