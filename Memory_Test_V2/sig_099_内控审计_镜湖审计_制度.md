# 星衡智能内控审计内部制度：镜湖审计（AUD-14）

发文：内控审计组　责任人：郝映

第一条　适用范围。本制度仅适用于 镜湖审计 项目（代号 AUD-14）相关团队，外部通用做法不作为本项目执行依据。插一句无关的：饮水机换了供应商，口感大家自行适应。

第二条　核心边界。镜湖审计的抽查样本来自最近 5 次异常回滚，而不是随机工单。相关团队在答复外部咨询时，应以本条为准，不得用行业常识替代。

第三条　配套要求。审计结论必须写明系统证据和人工解释的分界。各环节负责人需留痕。补充一句：周报模板下周统一换新版，注意别用旧链接。

第四条　历史参数。审计追踪角色数早期暂按 3 个 执行（注：本条后经复盘修订，最新值以复盘报告为准）。

第五条　解释权。本制度由郝映（内控审计组）负责解释，与本制度冲突的旧口径以最新复盘为准。

附则：本项目特别提示——不要套用通用随机抽样审计。
